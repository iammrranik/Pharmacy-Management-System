Pharmacy Management System
=========================

1. User (Member) Management
Base: /api/users

Fields: id, name, email, username, password, phone, role (ADMIN|CUSTOMER), createdDate

Action | Method | Endpoint | Sample JSON Body / Parameters
Add User | POST | /api/users | { "name": "Arif Ahmed", "email": "arif@mail.com", "username": "arif", "password": "pass123", "phone": "01711223344", "role": "CUSTOMER" }
Update User | PUT | /api/users | { "id": 1, "name": "Arif Rahman", "email": "arif.new@mail.com", "username": "arif", "password": "pass123", "phone": "01711223344", "role": "CUSTOMER" }
Get User | GET | /api/users/{id} | No body. Example: /api/users/1
Get User by Phone | GET | /api/users/phone/{phone} | No body. Example: /api/users/phone/01711223344
Get User by Username | GET | /api/users/username/{username} | No body. Example: /api/users/username/arif
List Users | GET | /api/users/page | Query: ?pageNo=1&pageSize=10
List All | GET | /api/users | No body. Returns list of all users
Count Users | GET | /api/users/count | No body. Returns integer
Delete User | DELETE | /api/users/{id} | No body. Example: /api/users/1
Delete by Phone | DELETE | /api/users/phone/{phone} | No body. Example: /api/users/phone/01711223344
Delete by Username | DELETE | /api/users/username/{username} | No body. Example: /api/users/username/arif

Base: /api/medicines
